> make icon-font from svgs


## Usage

```sh
  $ icon-font <input> [<output>]
```

## Options
  -f, --font-family  font name

## Examples

```sh
$ icon-font ./src/svg/\*.svg ./dest/iconfont --font-family=fontname
```
